## Install the project

Clone this repository :

```
git clone https://github.com/MoussaR84/AdevinaUnNumero
cd AdevinaUnNumero
```

Install packages :

```
yarn
```

When installation is complete, run it :


```bash
expo start
# or
yarn start
```
